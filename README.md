# LiliCAD Website

Static landing page for [LiliCAD](https://lilicad.com). All markup, styles, and scripts live in `index.html`, so you can edit everything from one file.

## Preview

```bash
python -m http.server 8000
```

Open `http://localhost:8000` and you're done.

## Deploy

Push to `main` and GitHub Pages serves the site as-is. Keep `CNAME` in the repo to preserve the `lilicad.com` domain.
